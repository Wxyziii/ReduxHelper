# ComfyUI Setup

ComfyUI is optional and local.

1. Start ComfyUI.
2. Confirm it is reachable at `http://127.0.0.1:8188`.
3. Open Settings → Image AI.
4. Set provider to ComfyUI.
5. Set checkpoint/model and sampler.
6. Generate from PNG previews only.

Generated PNGs are saved under the project workspace.
