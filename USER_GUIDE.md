# Redux AI Assistant User Guide

Redux AI Assistant helps you inspect and safely prepare Redux/GTA visual mod edits. It works on project-local workspace copies, validates AI suggestions, and creates manual export packages.

It does not install files into GTA, edit `.rpf` archives, or modify original files.

## Basic Flow
1. Create or open a project.
2. Import exported files or read-only archive extractions.
3. Scan files.
4. Search/select snippets.
5. Ask AI if desired.
6. Validate and review patch diffs.
7. Apply accepted patches to workspace copies.
8. Create export package.
9. Manually import files with OpenIV or relevant tools.
