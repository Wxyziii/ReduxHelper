# Troubleshooting

## OpenRouter key missing
Open Settings and add API key, or use non-AI workflows.

## texconv missing
DDS conversion requires texconv path in Settings.

## ComfyUI offline
Start ComfyUI and check URL.

## Invalid AI JSON
Ask AI again or paste a valid JSON response.

## Export validation failed
Open warnings and manifest. Missing files usually mean workspace output was moved or deleted.
