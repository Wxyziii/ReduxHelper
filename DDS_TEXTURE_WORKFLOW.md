# DDS Texture Workflow

1. Import DDS.
2. Inspect metadata.
3. Generate PNG preview with texconv.
4. Edit PNG manually or with ComfyUI.
5. Attach edited PNG.
6. Compare original vs edited with A/B slider.
7. Compile PNG back to DDS.
8. Mark compiled DDS export-ready.

AI never edits DDS directly. Original DDS files remain untouched.
